# CredentIQ- Fake News Detection Web Application

This project is a machine learning-based web application that classifies news as fake or real using NLP techniques.

## Features
- Real-time news prediction
- Machine learning model integration
- User-friendly web interface

## Technologies Used
- Python (Flask)
- HTML, CSS
- Machine Learning (NLP)
- SQLite

## How to Run
1. Install dependencies
2. Run app.py
3. Open in browser
